# 📧 Bulk Email Automation System

A full-stack application that enables sales teams to send personalized bulk emails using Excel data. Built with **React (frontend)** and **FastAPI (backend)**.

---

## 🚀 Features

- 📂 Upload Excel (.xlsx) files
- 📊 Automatic data parsing and validation
- ✉️ Send bulk emails with personalization
- ⚡ Background email processing (non-blocking)
- 📈 Scalable architecture
- 🔐 Secure environment variable handling

---

## 🧩 Tech Stack

### Frontend
- React (Vite)
- JavaScript / Axios / Fetch API

### Backend
- FastAPI
- Pandas (Excel processing)
- SMTP / Email API
